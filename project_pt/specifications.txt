
Все характеристики датасета. 

1.filename - имя файла, с записанным ответом ребенка

2. sex - пол

3. age - возраст

4. corpus - набор заданий (ENNY, Gillam, Conti4)

5. group - SLI (дети со специфическим расстройством языка), TD (условно нормально развивающиеся сверстники)

6.child_TNW - общее количество слов, произнесенных ребенком

7.child_TNS - общее количество предложений (дети с SLI чаще говорят простыми (короткими) предложениями).

8.examiner_TNW - общее количество слов, сказанных экзаменатором (дети с SLI чаще нуждаются в поддержке).

9.freq_ttr - соотношение токинов (общего количества слов) к типу слов. | Разделяет типы слов по токенам слов и дает приблизительную оценку лексического разнообразия.

10.r_2_iverbs - соотношение инфинитивов глаголов к измененяемой форме в соответствии с текстом (Дети с SLI часто испытывают трудности с морфемами -ed, -s, be и do. Это приводит к использованию инфинитивов глаголов вместо их изменяемых форм).

11.num_pos_tags - количество различных частей речи.

12.n_dos - количество раз, когда используется слово "do"

13.repetition - количество повторений одного слова, например, молоко молоко молоко молоко = молоко [x 4]

14.retracing - повторное отслеживаний, определяется как когда говорящий оставляет высказывание, но затем продолжает его снова.

15.fillers - подсчитывает общее количество использованных междометий. Список междометий был создан путем поиска по всему корпусу (все 1038 образцов) всех распространенных вариантов наполнителей, таких как um, umm, uh, uhh и т.д.

16.s1gppl - затруднения грамматики-1 в группе SLI  (Сложность этой выборки по сравнению с языковой моделью, обученной на всей группе SLI для этого корпуса, кроме образца)

17.s2gppl - затруднения грамматики-2 в группе SLI (то же, что и выше, но с 2-грамматической языковой моделью)

18.s3gppl - затруднения грамматики-3 в группе SLI (то же, что и выше, но с 3-грамматической языковой моделью)

19.d1gppl -затруднения грамматики-1 в группе TD  (Сложность этого образца по сравнению с языковой моделью, обученной на всей группе TD для этого корпуса, кроме образца).

20.d2gppl -  затруднения грамматики-2 в группе TD (то же, что и выше, но с 2-грамматической языковой моделью)

21.average_syl - среднее количество слогов в слове.

MLU - Средняя длина высказывания (или MLU) является показателем языковой продуктивности у детей. Традиционно он рассчитывается путем сбора 100 произнесенных ребенком высказываний и деления количества морфем на количество высказываний. Более высокий уровень MLU означает более высокий уровень владения языком. Про mlu https://en.wikipedia.org/wiki/Mean_length_of_utterance.

22.mlu_words - Средняя длина произнесения слов. 

23.mlu100_utts - Средняя длина произнесения первых 100 слов 

24.verb_utt - Количество высказываний, состоящих из глаголов

25.dss - Оценка меры сложности предложения

26.ipsyn_total - Индекс продуктивного синтаксиса (оценка сложности текста)

Следующие критерии представляют собой количество речевых форм, соответствующих стадиям морфологического развития Каролины Браун (см. Https://www.speech-language-therapy.com/index.php?option=com_content&view=article&id=33:brown&catid=2:uncategorised&Itemid=117)

Пункты 27-40.

27.present_progressive - использование глаголов в настоящей, действующей в этот момент форме (глагол c -ing)

28.propositions_in - количество предложений с предлогом 'in'

29.propositions_on - количеств предложений с предлогом 'on'

30.plurals - использование множественного числа

31.irregular_past_tense - количество неправильных глаголов прошедшего времени (me fell down) 

32.possessive_s - количество слов с "s" (обозначение чьей-то собственности - man's book)

33.uncontractible_copula - количество несокращаемых связок (полная форма глагола быть, когда это единственный глагол в предложении) Is it Alison?

34.articles - количество артиклей (a, the, etc.)

35.regular_past_ed - количество глаголов обычного прошедшего времени.

36.regular_3rd_persons - количество глаголов третьего лица в настоящем времени.

37.irregular_3rd_person - колчество нерегулярных глаголов третьего лица.

38.uncontractible_aux - количество полных форм глагола «быть», когда он является вспомогательным глаголом в предложении

39.contractible_copula - колчество сокращенных форм глагола "быть", когда это единственный глагол в предложении

40.contractible_aux - количество сокращенных форм глагола «быть», когда он является вспомогательным глаголом в предложении

41.Worderrors - количество ошибок в словах

42.fk - Результат Флеша-Кинкейда. См. https://en.wikipedia.org/wiki/Flesch%E2%80%93Kincaid_readability_tests

43.n_v - количество существительных, за которыми сразу следует глагол

44.n_aux - количество существительных, за которыми сразу следует вспомогательный глагол

45.n_3s_v - число существительных третьего лица единственного числа, за которыми сразу следует глагол

46.det_np_l - количество определяющих существительных, за которыми следует личное местоимение 

47.det_pl_n - количество определяющих местоимений, за которыми следует существительное

48.pro_aux - местоимения, за которыми следует вспомогательный глагол

49.pro_3s_v - количество местоимений 3-его лица им. падежа ед. числа, за которым следует Глагол 

50.total_error - общее количество морфосинтаксических ошибок (п.43-49)
